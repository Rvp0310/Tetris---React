# Tetris Next.js Project

A simple Tetris game built with **Next.js**, **React**, and **TypeScript**.  

## Features (WIP)
- Grid-based Tetris stage
- Tetromino shapes with colors
- Custom hooks for game logic
- Responsive layout using CSS Grid

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS (optional)
- Custom hooks for game logic
